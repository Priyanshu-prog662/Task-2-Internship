# Task-2-Internship

Task 2: Intermediate HTML, CSS, and JavaScript
This project covers intermediate web development concepts focusing on responsive layout models, JavaScript DOM manipulation, and input form validations. Built as part of the 45-day AICTE Web Development Internship.

🚀 Features
Validated Contact Form: A clean input form featuring text, email, and message fields with dynamic JavaScript validation (checks for empty fields and correct email regex structure) displaying inline errors.
Responsive Grid & Flexbox Layout: A dedicated page demonstrating structural styling using a Flexbox sticky navigation bar and a CSS Grid content showcase that adapts across desktop, tablet, and mobile views.
Dynamic To-Do List: An interactive task tracker application allowing users to add, toggle completed states, and remove list items dynamically from the page.
🛠️ Technologies Used
HTML5 (Forms & Sections)
CSS3 (Grid, Flexbox, Media Queries)
JavaScript (DOM Manipulation, Events, RegExp Validation)
📂 File Structure
index.html — Portfolio page containing the Contact Form section.
style.css — Layout styling, input focus transitions, and validation errors display.
script.js — Client-side input validation and error handlers.
layout.html — Responsive Grid and Flexbox demonstration page.
layout.css — Styling rules and column spans for the Grid demo.
todo.html — Dynamic To-Do List layout.
todo.css — Checklist item alignments and check status indicators.
todo.js — Task array state loops and list re-rendering logic.
